#  Здесь лежат моя конфигурация емакса и прочего говна... 
## Установка

``` console
user@PC: ~ $ git clone https://github.com/rasongame/dots
user@PC: ~ $ cd dots
user@PC: ~ $ cp .emacs.d ~/.emacs.d
user@PC: ~ $ cd .. && rm dots
```
